# Encounter Balance

Roll a d10 to determine the encounter difficulty:

| d10 Roll | Challenge Rating   |
| -------- | ------------------ |
| 1        | Party level - 1d4  |
| 2        | Party level - 1    |
| 3-5      | Party level        |
| 6        | Party level + 1    |
| 7        | Party level + 1d4  |
| 8        | Party level + 1d6  |
| 9        | Party level + 1d8  |
| 10       | Party level + 1d10 |
